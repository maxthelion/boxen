# Boxen: 3D Laser-Cut Box Designer

Boxen is a web-based application for designing and customizing boxes for laser cutting. It provides a 3D interface to visualize the box as you design it, and allows you to export the design as an SVG file that can be used with a laser cutter.

## Features

*   **3D Preview**: See your box design in 3D as you edit it.
*   **Hierarchical Component Tree**: View and manage the components of your box in a tree structure.
*   **Customizable Dimensions**: Easily change the dimensions of your box.
*   **Panel and Assembly Properties**: Fine-tune the properties of each panel and assembly.
*   **SVG Export**: Export your design as an SVG file for use with a laser cutter.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

*   Node.js and npm (or a compatible package manager)

### Installation

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username/boxen.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Start the development server
    ```sh
    npm run dev
    ```

## Available Scripts

In the project directory, you can run:

*   `npm run dev`: Runs the app in the development mode. Open [http://localhost:5173](http://localhost:5173) to view it in the browser.
*   `npm run build`: Builds the app for production to the `dist` folder.
*   `npm run preview`: Serves the production build locally.
*   `npm run lint`: Lints the code using ESLint.
*   `npm run typecheck`: Type-checks the TypeScript code.

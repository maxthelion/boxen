# Agent Recommendations

This directory contains recommendations from agent work sessions for future improvements that are outside the scope of the immediate task.

## Sections

| Directory | Purpose |
|-----------|---------|
| `testing/` | Testing gaps, missing test categories, test infrastructure improvements |
| `architecture/` | Design improvements, new abstractions, structural changes |
| `refactoring/` | Code cleanup, deduplication, consolidation opportunities |

## File Naming

Use format: `YYYY-MM-DD-short-description.md`

## Template

```markdown
# [Category]: Short Title

**Date:** YYYY-MM-DD
**Source:** [Task/audit/bug fix that surfaced this]
**Status:** Open | In Progress | Implemented | Rejected

## Problem

[What issue or opportunity was identified]

## Recommendation

[What should be done]

## Related Files

[Key files involved]

## Notes

[Any additional context]
```

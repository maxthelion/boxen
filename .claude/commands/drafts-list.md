# /drafts-list - List All Drafts

Display all draft documents with their status and age.

## Instructions

Run the list drafts script and display the output:

```bash
orchestrator/venv/bin/python .orchestrator/scripts/list_drafts.py
```

Show the full output to the user. If they ask about a specific draft, read it and summarize.

## Related Commands

- `/draft-idea` - Capture a new idea as a draft
- `/process-draft` - Process a draft into action (archive, extract rules, create tasks)
